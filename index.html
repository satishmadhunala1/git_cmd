basis commands

    
        Basic Git Commands
Configuration

sh
Copy code
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
These commands set your Git username and email address, which will be associated with your commits.

Initialize a Repository

sh
Copy code
git init
This command initializes a new Git repository in your current directory.

Clone a Repository

sh
Copy code
git clone 
This command clones an existing repository from a remote URL to your local machine.

Check Repository Status

sh
Copy code
git status
This command shows the status of your working directory and staging area, including any changes that have been staged, changes that have not been staged, and files that are not being tracked by Git.

Stage Changes

sh
Copy code
git add 
git add .
These commands add changes in files to the staging area. The first command stages a specific file, while the second stages all changes in the current directory.

Commit Changes

sh
Copy code
git commit -m "Commit message"
This command commits the staged changes to the repository with a message describing what was changed.

View Commit History

sh
Copy code
git log
This command shows the commit history for the repository.

View Differences

sh
Copy code
git diff
git diff --staged
The first command shows changes between the working directory and the staging area. The second command shows changes between the staging area and the last commit.

Branching and Merging
Create a New Branch

sh
Copy code
git branch 
This command creates a new branch called .

List Branches

sh
Copy code
git branch
This command lists all branches in the repository and shows the current branch.

Switch Branches

sh
Copy code
git checkout 
This command switches to the specified branch.

Create and Switch to a New Branch

sh
Copy code
git checkout -b 
This command creates a new branch and switches to it immediately.

Merge Branches

sh
Copy code
git merge 
This command merges the specified branch into the current branch.

Delete a Branch

sh
Copy code
git branch -d 
This command deletes the specified branch.

Remote Repositories
Add a Remote Repository

sh
Copy code
git remote add origin 
This command adds a remote repository and names it origin.

View Remote Repositories

sh
Copy code
git remote -v
This command lists the remote repositories configured for the local repository.

Fetch Changes from a Remote Repository

sh
Copy code
git fetch
This command fetches changes from the remote repository but does not merge them into your working directory.

Push Changes to a Remote Repository

sh
Copy code
git push origin 
This command pushes the changes from your local branch to the corresponding remote branch.

Pull Changes from a Remote Repository

sh
Copy code
git pull
This command fetches and merges changes from the remote repository into your current branch.

Undoing Changes
Unstage Changes

sh
Copy code
git reset 
This command removes the specified file from the staging area but leaves the working directory unchanged.

Revert Changes in Working Directory

sh
Copy code
git checkout -- 
This command discards changes in the working directory for the specified file.

Reset to a Previous Commit

sh
Copy code
git reset --hard 
This command resets the working directory and staging area to match the specified commit, discarding all changes since that commit.

Revert a Commit

sh
Copy code
git revert 
This command creates a new commit that undoes the changes from the specified commit.

Advanced Commands
Stash Changes

sh
Copy code
git stash
This command temporarily saves changes in the working directory that are not ready to be committed.

Apply Stashed Changes

sh
Copy code
git stash apply
This command applies the most recently stashed changes to the working directory.

List Stashes

sh
Copy code
git stash list
This command lists all stashes that you have saved.
